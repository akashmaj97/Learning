
Step1: Right click on the application that you want to make as a startup application.

Step2: Go for the "property" option and copy the target path from there ( or Exact location of that Application/exe).

Step3: Now, click on window button and search for "Run"

Step4: Then type "SHELL:STARTUP" and press enter

Step5: Below path will get open 
        C:\Users\Admin\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup

Step6: Right Click on that directory and go for "new" and then "shorcut".

Step7: A pop up eill open asking for the path of the application which you want to run.

Step8: Copy the target path that we got in step 2 and paste it on the pop up.
